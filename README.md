# szsh
speedie's zsh configuration.
Works well with my [st](https://github.com/speedie-de/st) configuration.

Installation
- git clone <this-page>
- cd szsh/.config
- cp -r zsh ~/.config
- echo "export ZDOTDIR=~/.config/zsh/dotfiles" >> ~/.zprofile
- Install zsh using your package manager
- chsh /bin/zsh
